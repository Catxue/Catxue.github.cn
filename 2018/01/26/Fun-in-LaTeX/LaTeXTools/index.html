如何在Sublime Text中配置LaTeXTools

之前我的博客已经推荐了几款比较好用的 $\LaTeX$ 编辑器，其中提到的 LaTeXTools 是 GitHub 上为 Sublime Text 软件开发的一款插件，我使用了一段时间感觉很好用。这个插件美中不足的一点就是配置起来不是很方便，所以我这里给大家讲解一下如何在 Sublime Text 中配置 LaTeXTools。

准备工作

  1. 确认电脑是否安装 Sublime Text, 以及确认一下版本是 SublimeText 2 还是 3.
  2. 确认电脑安装 $\LaTeX$ 版本，是 TeXLive 还是 MiKTeX.(没安装的童鞋推荐 TeXLive)

安装LaTeXTools

  • 在 Sublime Text 中安装 Package Control 。（如果已经安装过了可以跳过这一步）

    按下 ctrl + ` ,或 View > Show Console,然后将下面的代码复制粘贴进去，然后按下回车

Sublime text 3

Sublime text 2

安装好了之后看以看到下面的选项：Preference->Package Control

  • 确认 Package Control已经安装完成后，按下组合键 Ctrl+Shift+P，然后再输入 install，选择 Package Control: install package。进入库后，搜索LaTeXTools, 选择安装库就可以了。

配置LaTeXTools

前面所有的工作只是把LaTeXTools安装成功，接下来最为关键的是如何配置LaTeXTools。

配置$\LaTeX$编译器
首先在Sublime中依次选择 Preferences | Package Settings | LaTeXTools | Settings – User ，然后进入LaTeXTools设置菜单。

如下图所示，将本地编译器地址配置到texpath选项，然后在distro中匹配$\LaTeX$版本。sumatra配置可以不填写。

现在LaTeXTools基本配置好了，按下ctrl + B后 Sublime 可以调用LaTeXTools的build进行编译了。

进阶配置

配置SumatraPDF

对于很多喜欢$\LaTeX$的用户，需要在PDF中双击相应的内容，直接定位到Sublime Text 中的$\LaTeX$的内容，这样就可以极大地提高工作效率，因此需要配置一下反向搜索。这里我们推荐使用 Sumatra PDF 阅读器来预览PDF文件，这是一款轻量级并且支持正向和反向搜索的小众软件。

一旦你程序安装完毕后，你需要添加一个地址让LaTeXTools可以轻松找到它。

1. 添加 SumatraPDF 的主程序目录添加到环境变量PATH

这里有两种方式添加环境变量，推荐在命令行中cmd.exe中使用命令行进行添加

set PATH %PATH%;C:\Program Files\SumatraPDF

这里C:\Program Files\SumatraPDF替换为你自己安装SumatraPDF的地址。
另一种就是在电脑环境变量中手动添加。

2. 配置SumatraPDF反向搜索
配置SumatraPDF反向搜索同样有两种方式，在命令行中执行以下命令

sumatrapdf.exe -inverse-search "\"C:\Program Files\Sublime Text 3\sublime_text.exe\" \"%f:%l\""

上述路径要替换成自己软件安装路径。
如果觉得命令行方式不够友好，你可以找之前带有对应.synctex.gz文件的PDF文件，然后打开此PDF文件，在 Settings | Options中输入一下命令

  • (ST3) "C:\Program Files\Sublime Text 3\sublime_text.exe" "%f:%l"
  • (ST2) "C:\Program Files\Sublime Text 2\sublime_text.exe" "%f:%l"

同样你也要把上述安装地址换成自己安装的目录，如下图所示。

配置公式预览

如果你安装的是 Sublime Text 3 version 3118和在此之后的版本，都可以使用公式预览这个功能。这里你需要确认你电脑中是否安装了 Ghostscript 8 及其以上的版本，并且可以在设置的路径中找到这个程序。

这里你可以使用 LaTeXTools: Check System 命令检查你程序的安装路径以及版本。
