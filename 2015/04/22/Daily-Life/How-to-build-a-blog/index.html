建立博客已经很长时间了，一直没有写一下建立博客的资料，或许也称之为“攻略”吧。写毕设论文写得头疼，特闲暇之余总结一下如何利用Hexo建立博客的资料，希望对读者有所启发。

什么是博客

什么是博客呢？Blog的English解释是Web Log，意指log on the web。我认为博客就是一个可以记录生活、抒发情感也可以分享学习知识、结交朋友的平台。当然对于单身男士们，博客是一个很好地提升自己逼格和把妹的平台，有博客就是这么任性！废话少说，接下来就来介绍如何在短期内很好地建立自己的博客。

Git

什么是Git,看下面的官方定义：

Git is a free and open source distributed version control system
designed to handle everything from small to very large projects with
speed and efficiency.

看了这个你也许会想,纳尼！建立一个博客还要学习什么Git语言，不写博客也罢。少年~，不要着急，这个只要求入门级别就行，大概用个半天时间就足够了。学习Git只是让你知道如何建立远程仓库以及上传自己的博客而已，当然如果你想学得很nice，那就要花个几天了。

Git入门我推荐你学习廖雪峰写的史上最浅显易懂的Git教程。如果你觉得不够那就看看图解Git，当然你有强迫症，非要搞得很透彻，那就看这个Pro Git吧。

hexo

什么是hexo,看看官方定义：

A fast, simple & powerful blog framework

hexo出自台湾大学生Tommy Chen之手，是一个基于Node.js的静态博客程序，其编译上百篇文字只需要几秒。当然有人也用Octopress搭建博客，那就还是看看Hexo 颯爽登場！就知道两者的区别了。hexo需要了解什么呢？仅仅需要了解下面几条命令即可。

安装hexo

npm install hexo-cli -g

初始化

hexo init <folder>

安装npm插件

npm install

安装hexo-deployer-git插件，此插件会在主目录下生成.deploy_git文件并强行按照_config中配置进行推送。如果不存在.deploy_git文件，可通过git init命令建立。

npm install hexo-deployer-git --save

其中配置选项为：

deploy:
  type: git
  repo: <repository url>
  branch: [branch]
  message: [message]

克隆主题，在博客根目录下执行clone命令，例如

 git clone https://github.com/wuchong/jacman.git themes/jacman

启用
修改你的博客根目录下的_config.yml配置文件中的theme属性，将其设置为jacman。同时建议设置stylus属性中的compress值为true，会自动压缩 CSS 文件，hexo默认配置中不包含这一项，建议开启。如下。

theme: jacman
stylus:
  compress: true

建立新文章

hexo new_post <title>

生成博客文件

hexo generate

上传服务器

hexo server

清除文件

hexo clean

hexo常用的命令就这么几条，是不是很easy？ O(∩_∩)O~~

Markdown

Markdown是什么东东？请看下面：

Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。

Markdown相对于每一个看此篇博文的人都是件很Easy的事情，为什么这样说呢？因为我仅仅用了十分钟就学会了。如果你还是学不会而且还是个温柔贤惠的妹子，可以选择联系我哦。

关于Markdown网上的资源有很多，我推荐看三篇文档

  1. Markdown 语法说明 (简体中文版)
  2. 作业部落
  3. Markdown

其实我是很推荐第三篇文档的电子书的，只是没有找到免费电子版的，考虑到不是所有的人都会花钱去看，就姑且放在最后了。在这里还是想吐槽一下那些觉得花几块钱买本书看都要BB半天的人，如果说你可以买得起iPad却看不起几本几十元的书，请自觉绕开，我们不是一个次元。

HTML和CSS

这部分不是必学的，只是推荐而已。学过这个以后可以自己定制自己修改博客的样式当然也可以尝试自己写写博客模板。学习HTML和CSS后你就是半个前端工程师了，想到这里你是想学还是想学啊？推荐在W3School网站学习HTML和CSS,当然作为进阶使用，你可以顺便学习一下XML。

如果真心不想学习那么多东西，那还是推荐抽出时间看一下HTML 30分钟入门教程。至少有个网页的基本概念，以及了解一些转义之类的东西。
